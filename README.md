# Detection-car-exterior-components
Hello world this project is for a deep learning challenge. I hope you find it helpful <3 <br/>
This project is an exterior car parts detection
<br/>
We have 20 classes <br/>
    -Front Bumper <br/>
    -Rear Bumper <br/>
    -LT Fender	<br/>
    -RT Fender	<br/>
    -LT Headlight <br/>
    -RT Headlight <br/>
    -LT Taillight <br/>
    -RT Taillight	<br/>
    -Hood			<br/>
    -Grill			<br/>
    -Windshield		<br/>
    -Mirror			<br/>
    -LT Front Door	<br/>
    -RT Front Door	<br/>
    -LT Rear Door	<br/>	
    -RT Rear Door	<br/>	
    -LT Quarter Panel <br/>	
    -RT Quarter Panel <br/>
    -Trunk			<br/>
    -Wheel			<br/>
